# AI Web Hub

Minimal, fast and open source power-user web navigator.

## Goals

- Fast navigation between key sites
- Keyboard-first UX
- Lightweight footprint
- Optional AI summarization

## Target Sites

- YouTube
- ChatGPT
- Gemini
- Claude
- Linkedin
- Amazon
- Gitlab
- Reddit
- GitHub
- X (Twitter)
- Gmail
- Wikipedia

## Tech Stack

- Tauri (desktop shell)
- TypeScript frontend
- Optional Python AI service

## Status

MVP in progress.
